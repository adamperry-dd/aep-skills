# DD Branding

This Skill provides comprehensive DD brand guidelines across multiple domains to ensure consistent brand representation in all materials.

## Quick start

When creating any DD-branded content, consult the relevant domain reference below. Each domain contains specific guidelines, examples, and best practices.

## Available domains

Choose the domain most relevant to your current task:

**Voice & Tone**: Messaging style, communication guidelines, brand personality → See [reference/voice-and-tone.md](reference/voice-and-tone.md)

**Visual Identity**: Logo assets, colors (hex: #FF3008, gradients), TT Norms Pro typography, file formats → See [reference/visual-identity.md](reference/visual-identity.md)

**Messaging Framework**: Key messages, value propositions, "Your Door To More" campaign → See [reference/messaging.md](reference/messaging.md)

**Content Guidelines**: Writing style, formatting standards, content types → See [reference/content-guidelines.md](reference/content-guidelines.md)

**Co-Marketing & Partnerships**: Logo lockups, partnership guidelines, co-branded materials → See [reference/co-marketing-partnerships.md](reference/co-marketing-partnerships.md)

## General principles

1. **Consistency**: All branded materials should feel cohesive across channels
2. **Clarity**: Communicate clearly and avoid jargon unless audience-appropriate
3. **Authenticity**: Maintain genuine voice that reflects DD values
4. **Quality**: All materials should meet professional standards

## When to use this Skill

Use this Skill when:
- Creating presentations, documents, or marketing materials
- Writing content for DoorDash websites, blogs, or social media
- Designing branded assets or visual materials
- Working on co-branded or partnership materials
- Selecting logo files and formats (RGB, CMYK, SVG, PNG, etc.)
- Ensuring consistency with DoorDash brand standards
- Questions about proper logo usage, colors, or typography
- Developing messaging for new products or features
- Creating logo lockups with partner brands
- Working on "Your Door To More" campaign materials

## Navigation tips

Start with the general principles above, then dive into the specific domain reference that matches your task. Each reference file contains detailed guidelines with concrete examples.
